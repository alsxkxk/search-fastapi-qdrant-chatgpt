## 👨‍💻 프로젝트 소개 
ChatGPT를 기반으로 가상 기술면접 서비스를 제공합니다. </br> </br>
기술면접을 준비하며 여러가지 서비스를 이용해보았지만

|서비스명|단점|
|------|---|
|데브톡|UI / UX는 좋았지만 서비스를 더이상 이용하지 못함|
|테크톡|꼬리물기 질문 불가능, 할루시네이션 발생|
|FLYIT|단순 문제은행 서비스|

위와같은 불편한점들을 느끼게 되었고 직접 만들어보고자 CSpeak을 개발하게 되었습니다.
</br></br>
